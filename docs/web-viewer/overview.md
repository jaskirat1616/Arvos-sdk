# Web Viewer Overview

Zero-install browser-based viewer for ARVOS sensor data.

## Features

- ✅ **No Python Required** - Works in any browser
- ✅ **30 Second Setup** - Start in under a minute
- ✅ **Live Visualization** - Real-time sensor data display
- ✅ **Works Everywhere** - Any device with a browser

## Quick Start

```bash
cd arvos-sdk/web-viewer
./start-viewer.sh    # Mac/Linux
# OR
start-viewer.bat     # Windows
```

Then scan the QR code with your iPhone!

[→ Quick Start Guide](quick-start.md)

## What It Shows

- **Camera Feed** - Live RGB video
- **Point Cloud** - 3D LiDAR visualization
- **IMU Data** - Accelerometer and gyroscope
- **GPS** - Location on map
- **Pose** - ARKit tracking visualization

## Use Cases

- Quick testing without SDK setup
- Live demos and presentations
- Students and researchers
- Any device with a browser
- Zero-install scenarios
