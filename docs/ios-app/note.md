# iOS App Source Code

## App Availability

The **ARVOS iOS app** is available for download and use on the App Store. The app provides a full-featured sensor streaming platform for iPhone and Apple Watch.

## Source Code Status

!!! important "Private Repository"
    The iOS app source code is **not open source** and is **not publicly available** on GitHub. The app is distributed through the App Store as a compiled binary.

### What This Means

- ✅ **App is available** - Download and use the app from the App Store
- ✅ **Full functionality** - All features work as documented
- ✅ **SDK is open source** - Python SDK code is fully open source
- ❌ **App code is private** - iOS app source code is not available

### Why Keep It Private?

The iOS app source code is kept private for:
- **Intellectual property protection**
- **Commercial considerations**
- **Code quality control**
- **Security and stability**

### What You Can Do

Even though the app code is private, you can:
- ✅ Use the app fully (all features work)
- ✅ Contribute to the Python SDK (fully open source)
- ✅ Report bugs and request features
- ✅ Use all protocols and features
- ✅ Integrate with your own systems

### Contributing

If you want to contribute:
- **Python SDK**: Fully open source - contributions welcome!
- **Documentation**: Help improve docs
- **Examples**: Add more examples
- **Bug Reports**: Report issues you find
